
## 멀티쓰레드 및 객체지향 설계
---

### 프로젝트 소개
##### 멀티 쓰레드에 관련한 테스트 코드 작성과 객체지향 관점의 설계로 구현한 프로그램입니다.
---

### 사용 스택
+ Spring Boot
+ InMemory(Map Collection)
+ Lombok
+ Junit5


---
### 프로그램 실행

```
java -jar /out/artifacts/~.jar
```
